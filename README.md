### Python environment setup with Conda
```bash
conda create -n draGAN python=3.7 anaconda
conda activate draGAN

conda install pytorch=1.13 torchvision torchaudio -c pytorch -c nvidia
pip install imbalanced_databases

conda clean --all
```
